# Custom MCP Server

Sandboxed file workspace MCP server used by the guarded agent demo.

## Tools
- `list_files`
- `read_file`
- `write_file`
- `delete_file`
- `search_files`

## Run
- `python -m armoriq_mcp.server`

## Sandbox Root
- Defaults to `mcp_server/sandbox`
- Override with `ARMORIQ_SANDBOX_ROOT`
